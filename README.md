# Advanced React Architecture & State Management

This project serves as a practical application and comprehensive review of advanced React concepts, with a strong emphasis on clean code architecture and scalable state management. 

## 🚀 Project Overview

Built using modern React and Vite, this application demonstrates how to handle complex global state efficiently without relying on third-party libraries like Redux. The primary goal of this repository is to showcase maintainable, production-ready React patterns.

## 🧠 Key Concepts & Features

* **Advanced State Management:** Implemented a robust global state solution using a combination of `useReducer` and `useContext`, providing a predictable state container and avoiding prop drilling.
* **Clean Code Principles:** Focused on writing **Pure Components** to ensure predictable rendering, reusability, and easier testing. The codebase is structured to separate business logic from UI components.
* **Modern Build Tooling:** Scaffolded with [Vite](https://vitejs.dev/) to leverage blazing-fast Hot Module Replacement (HMR) and optimized build performance.

## 🛠️ Tech Stack

* **Core:** React
* **State Management:** Context API + `useReducer`
* **Build Tool:** Vite (React + SWC/Babel)

## ⚡ Getting Started

To run this project locally:

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
